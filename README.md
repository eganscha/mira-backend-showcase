# Mira Backend

## Project Description

Mira is a marketplace platform for local services that connects customers with verified service providers. This repository contains the Java backend for the platform, built with Spring Boot, Spring Security, PostgreSQL and JPA.

The backend focuses on REST oriented API design, OAuth2/OpenID Connect based login, JWT based authentication and structured API documentation with OpenAPI.

Planned features include Stripe based payment processing and AI assisted accessibility features using LLM and VLM integration.
